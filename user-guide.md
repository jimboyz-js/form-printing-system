# FPS User Guide

## Introduction

FPS (Form Printing System) is a desktop application that helps users generate and print structured documents such as Billing Statements and Charge Invoices.

## Main Menu

### File Menu
- Home
- Print
- Exit

### System Menu
- Billing Statement
- Charge Invoice

### Help Menu
- User Guide
- About
- Preferences

## Clearing Input Fields

Press:

CTRL + C

to clear all input fields.

## Preferences

The Preferences menu allows customization of:

- Calendar settings
- User interface themes
- Font settings

## Printing Documents

1. Enter the required information in the form fields.
2. Select the document type (Billing Statement or Charge Invoice).
3. Click **Print** or press **CTRL + P**.

The system will generate a formatted printable document using JasperReports.
